## ‚ö° Instalasi Cepat

Pastikan Anda memiliki PHP 8.2+, Composer, dan Node.js.

1. **Clone & Setup**:
   ```bash
   git clone https://github.com/RacoonHQ/conq.git
   cd laravel-conq
   composer run setup
   ```

2. **Jalankan Server**:
   ```bash
   composer run dev
   ```
   Akses aplikasi di [http://localhost:8000](http://localhost:8000).
